# Contributing to ProfileBase

Thank you for your interest in contributing to ProfileBase! We welcome contributions from the community to help improve this project.

## Code of Conduct

Please be respectful and considerate of others when contributing. We strive to maintain a welcoming and inclusive environment.

## How to Contribute

1.  **Fork the Repository:** Create a fork of the repository to your own GitHub account.
2.  **Clone the Repository:** Clone your fork to your local machine.
    ```bash
    git clone https://github.com/your-username/profilebase.git
    cd profilebase
    ```
3.  **Create a Branch:** Create a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Install Dependencies:**
    ```bash
    npm install
    # or
    pnpm install
    ```
5.  **Make Changes:** Implement your feature or fix. Ensure your code follows the project's coding standards.
6.  **Test Your Changes:** Run the development server and verify your changes.
    ```bash
    npm run dev
    ```
7.  **Commit Your Changes:** Commit your changes with clear and descriptive commit messages.
    ```bash
    git commit -m "feat: Add amazing feature"
    ```
8.  **Push to Your Fork:** Push your branch to your forked repository.
    ```bash
    git push origin feature/amazing-feature
    ```
9.  **Submit a Pull Request:** Open a Pull Request (PR) from your fork to the main repository. Provide a clear description of your changes.

## Development Guidelines

- **Tech Stack:** Next.js, Tailwind CSS, Prisma, TypeScript.
- **Styling:** Use Tailwind CSS for styling.
- **Icons:** Use [Tabler Icons](https://tabler-icons.io/) (`@tabler/icons-react`).
- **Components:** Keep components small, focused, and reusable.
- **Type Safety:** Ensure strict type checking with TypeScript.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible to help us understand and resolve the issue.

Thank you for contributing!
